# SistemaLivraria

Sistema desenvolvido para o gerenciamento de livros incluindo inserção, atualização, edição, remoção e busca.

-- O Script do Banco encontra-se em DatabaseScript.

-- Tecnologias Utilizadas:

		- Web API
		- MVC
		- JavaScript
		- Sql Server 2014
		- Jquery
		- Ajax
		- BootStrap
